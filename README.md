# Birthright-Genesys

Conversion of the **AD&D Birthright** campaign setting for the **Genesys RPG**.  
Reimagines divine bloodlines, domain rulership, and political intrigue using Genesys’ narrative dice system.  
Written entirely in Markdown for modular rules, lore, and future PDF publication.

---

## Repository Structure
- **/docs** — Markdown rulebooks and conversion documents  
- **/assets** — Images, maps, tables, and supporting materials  
- **/tools** — Utilities or reference files for content management  
- **/.github** — Configuration, workflows, and Copilot instructions  

---

## Project Status
This repository is in the **initial setup phase**.  
Future commits will include:
- Detailed Markdown rulebooks  
- Conversion documents for Birthright mechanics and lore  
- Reference materials and assets  
- Tools for content and rule management  

---

## AI Collaboration Guidelines
This project uses **GitHub Copilot** and **ChatGPT** for structured document generation and conversion assistance.  
All AI interactions follow the standards defined in  
[`.github/copilot-instructions.md`](.github/copilot-instructions.md).

**Purpose:**  
Ensure consistent tone, formatting, and alignment with Genesys design principles while converting AD&D Birthright material.  

**Reminder:**  
AI output must never reproduce copyrighted AD&D text—only original rewrites and system conversions.
