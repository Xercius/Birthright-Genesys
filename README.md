# Birthright-Genesys

Conversion of the AD&D Birthright campaign setting for the Genesys RPG. Reimagines divine bloodlines, domain rulership, and political intrigue using Genesys' narrative dice system. Written in Markdown for modular rules, lore, and future PDF publication.

## Repository Structure

- `/docs` - Markdown rulebooks and conversion documents
- `/assets` - Images, tables, and supporting materials
- `/tools` - Scripts and utilities for content generation
- `/.github` - GitHub configuration and workflows

## Project Status

This repository is currently in the initial setup phase. Future commits will add:
- Detailed Markdown rulebooks
- Conversion documents for Birthright content
- Reference materials and assets
- Tools for content management
