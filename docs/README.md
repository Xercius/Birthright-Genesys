# Documentation

This folder contains the Markdown rulebooks and conversion documents for the Birthright-Genesys project.

## Structure
Documentation will be organized into logical sections covering:
- Core rules and mechanics
- Bloodline abilities and divine powers
- Domain management and rulership
- Political intrigue and warfare
- Character creation and advancement
- Setting lore and locations

## Format
All documents are written in Markdown for easy editing and future PDF compilation.
